# Images

Ce dossier contiendra les images locales de l'application.

Pour l'instant, les images des produits sont chargées depuis Unsplash via des URLs.

Si vous souhaitez utiliser des images locales :
1. Placez vos images dans ce dossier
2. Mettez à jour le fichier `keyboards_mock.json`
3. Changez les URLs en chemins locaux : `assets/images/mon_image.png`
